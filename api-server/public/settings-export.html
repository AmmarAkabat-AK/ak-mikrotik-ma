متطلبات اتصال MikroTik

بروتوكول الاتصالRouterOS API (TCP Binary)
المنفذ الافتراضي8728 (بدون تشفير)
منفذ SSL8729 (مع تشفير TLS)
نوع الـ IP المطلوبPublic IP فقط أو DDNS
تفعيل API في MikroTikIP › Services › api › Enable
المكتبة المستخدمةnode-routeros@1.6.9
خطوات تفعيل API على MikroTik
#الخطوةالأمر / الطريق
1فتح Winbox أو WebFig—
2الذهاب لـ IP › ServicesIP → Services
3تفعيل apiapi → Enable ✓
4السماح بالمنفذ في FirewallIP → Firewall → Filter → Allow TCP 8728
5ضبط Port Forwarding (إن كان خلف NAT)IP → Firewall → NAT → dst-nat 8728
6التأكد من Public IP أو DDNSIP → DDNS أو مزود خدمة DDNS
